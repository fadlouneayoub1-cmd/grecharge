# grecharge (Recharge & SIM Manager)

A premium, responsive single-page web application designed for mobile and desktop devices to manage and record Telecom Recharge sales, SIM card transactions, Client debt ledgers, and operational expenses, backed by Supabase.

## Features
- **Mobile First Design**: Stays fixed at the bottom on mobile viewports for easy navigation.
- **Dynamic Localization**: Instantly toggle between French and Arabic across all interfaces, placeholders, tables, and system forms.
- **Arabic RTL Support**: True Right-to-Left text direction and column layout swapping.
- **Thermal Receipt Printing**: Print optimized tickets on 80mm/58mm thermal rolls.
- **WhatsApp Integration**: Share formatted ASCII text receipts (Bons) directly to Moroccan phone numbers.
- **QR Code Engine**: Generate crisp client QR cards and simulate camera scanners to trigger sales.
- **Profit Tracking**: Auto-calculates net profit margins based on buying commissions minus client discounts.
- **Supabase Integration**: Live Cloud-based synchronization of data.
